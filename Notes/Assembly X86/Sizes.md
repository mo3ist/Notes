#### [[Registers|Registers]] sizes 
![[Pasted image 20210217095313.png]]

- Lower 16 bits:
	- EAX -> AX
	- EBX -> BX
	- ECX -> CX
	- EDX -> DX
- The lower [16 - 8] bits:
	- EAX -> AH
	- EBX -> BH
	- ECX -> CH
	- EDX -> DH
- The lower 8 bits:
	- EAX -> AL
	- EBX -> BL
	- ECX -> CL
	- EDX -> DL

#### Data sizes
![[Pasted image 20210217095725.png]]

`byte ptr` -> 8 bits
`word ptr` -> 2 bytes (nibble)
`dword ptr` -> 4 bytes
`qword ptr` -> 8 bytes
